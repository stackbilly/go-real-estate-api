Real Estate API

Find Data for Your Real Estate Projects

Obtain Real Estate data for use in your applications and models. Data contains House Name, Description, Location, Price, Url, and Image. Use the API to query and receive data straight to your backend in json format. Contributions and improvements to the API are open and so feel free to provide suggestions via GitHub issues and pull requests.
